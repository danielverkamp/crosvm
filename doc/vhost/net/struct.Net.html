pub struct Net<T> {
    descriptor: File,
    phantom: PhantomData<T>,
}
Expand description

Handle to run VHOST_NET ioctls.

This provides a simple wrapper around a VHOST_NET file descriptor and methods that safely run ioctls on that file descriptor.
