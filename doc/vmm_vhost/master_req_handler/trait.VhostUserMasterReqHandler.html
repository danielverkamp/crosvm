Expand description

Define services provided by masters for the slave communication channel.

The vhost-user specification defines a slave communication channel, by which slaves could request services from masters. The VhostUserMasterReqHandler trait defines services provided by masters, and it’s used both on the master side and slave side.

  • on the slave side, a stub forwarder implementing VhostUserMasterReqHandler will proxy service requests to masters. The Slave is an example stub forwarder.
  • on the master side, the MasterReqHandler will forward service requests to a handler implementing VhostUserMasterReqHandler.

The VhostUserMasterReqHandler trait is design with interior mutability to improve performance for multi-threading.
