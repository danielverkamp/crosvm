Expand description

Represents a qcow2 file. This is a sparse file format maintained by the qemu project. Full documentation of the format can be found in the qemu repository.

Example

    let mut q = QcowFile::from(file, disk::MAX_NESTING_DEPTH).expect("Can't open qcow file");
    let mut buf = [0u8; 12];
    let mut vslice = VolatileSlice::new(&mut buf);
    q.read_at_volatile(vslice, 10)?;
