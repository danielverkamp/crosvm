Expand description

A trait for vhost-user devices.

Upon being given an [Executor], a device can be converted into a [VhostUserBackend], which can then process the requests from the front-end.

We don’t build VhostUserBackends directly because in the case of jailing, the device is built in the main process but it runs in the jailed child process. Since Executors cannot be passed to other processes, we cannot access the device’s executor at build time and thus need to perform this 2-step dance before we can run the vhost-user device jailed.
