Module base::descriptor_reflection::with_raw_descriptor

source · [−]
Expand description

Module that exports serialize/deserialize functions for use with #[serde(with = "...")] attribute. It only works with fields with RawDescriptor type.

Examples

use serde::{Deserialize, Serialize};
use base::platform::RawDescriptor;

#[derive(Serialize, Deserialize)]
struct RawContainer {
    #[serde(with = "base::platform::with_raw_descriptor")]
    rd: RawDescriptor,
}

Re-exports

pub use super::serialize_descriptor as serialize;
