pub unsafe fn build_fixed(
    self,
    addr: *mut u8
) -> Result<MemoryMapping, MmapError>

Build a MemoryMapping from the provided options at a fixed address. Note this is a separate function from build in order to isolate unsafe behavior.

Safety

Function should not be called before the caller unmaps any mmap’d regions already present at (addr..addr+size). If another MemoryMapping object holds the same address space, the destructors of those objects will conflict and the space could be unmapped while still in use.

WARNING: On windows, this is not compatible with from_file. TODO(b:230901659): Find a better way to enforce this warning in code.
